# 🎂 Hệ thống quản lý cửa hàng bánh bán bánh ngọt

Dự án xây dựng hệ thống quản lý cửa hàng bánh kem bằng ASP.NET Core MVC kết hợp MySQL nhằm hỗ trợ quản lý bán hàng, sản phẩm, hóa đơn và khách hàng một cách hiệu quả.

---

# 📌 Giới thiệu

Hệ thống giúp cửa hàng quản lý toàn bộ hoạt động bán bánh kem như:

- Quản lý sản phẩm
- Quản lý đơn hàng
- Quản lý khách hàng
- Quản lý nhân viên
- Thống kê doanh thu
- Theo dõi tình trạng đơn hàng

Dự án được thực hiện nhằm phục vụ môn Công nghệ phần mềm.

---

# ✨ Chức năng chính

## 👤 Quản lý tài khoản
- Đăng nhập / Đăng xuất
- Phân quyền người dùng

## 🍰 Quản lý sản phẩm
- Thêm sản phẩm
- Sửa thông tin sản phẩm
- Xóa sản phẩm
- Tìm kiếm sản phẩm

## 📦 Quản lý đơn hàng
- Tạo hóa đơn
- Chọn sản phẩm
- Cập nhật trạng thái đơn hàng
- Xem chi tiết hóa đơn

## 👥 Quản lý khách hàng
- Thêm khách hàng
- Tìm kiếm khách hàng
- Cập nhật thông tin khách hàng

## 📊 Thống kê
- Thống kê doanh thu
- Biểu đồ doanh thu
- Thống kê số lượng đơn hàng
